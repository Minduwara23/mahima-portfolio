# M.I. Weerapperuma — Portfolio

Personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## 🚀 Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save** — your site will be live at `https://<your-username>.github.io/<repo-name>/`

## 🎨 Color Theme

- Background: `#0d0d0f` (near black)
- Accent: `#a855f7` (purple)
- Text: `#ffffff` / `#d1d5db`

## 📁 Structure

```
├── index.html   # Main HTML
├── style.css    # All styles
├── script.js    # Scroll effects, nav, animations
└── README.md
```
